the-passing-of-time
===================
锦瑟
锦瑟无端五十弦,一弦一柱思华年.
庄生晓梦迷蝴蝶,望帝春心托杜鹃.
沧海月明珠有泪,蓝田日暖玉生烟.
此情可待成追忆,只是当时已惘然.

人之所以悲哀，是因为我们留不住岁月，更无法不承认，青春，有一日是要这么自然的消失过去，三毛如是说。青春和岁月不经意间就那么悄悄地溜走了，而我们却还不及细细体味其中滋味。那些逝去的日子，那些曾经的往事，无论悲喜哀愁都让我们难以忘怀。
人生如流水，有时平静有时汹涌，但总是去而不返，无能逆流。回忆往事，不禁感慨时光流逝，我们面对过往思绪万千，追悔莫及者、无可奈何者、无怨无悔者，不一而足。
作文一事，并非全搬以往事，或是非生活人物不写。故本文大略虚构，有其人其事者，无意于影射讽刺另有所指，笔者不过造些随意文字，自娱一番，编一二故事，下笔拙劣，行文不堪，也在所难免。文章主旨意在言情言事，读者自可揣摩。此开卷也。
9月的北京，烈日炎炎，天气燥热难当。立秋已过，时令快到秋分。但照目前的天气看，夏天虽已过去了，但秋天却还远远地不肯到来。西直门地铁站口人流攒动，熙熙攘攘，满是热量的因子，周围马路车站的拥挤和喧闹更给人平添了好多烦恼。而嘉茂大楼依旧矗立在那里俯视着眼下众生百象，不论白天黑夜它就站在那儿，不笑不怒不热不冷，即时看到在西直门桥上转晕掉了的车和人它也如此。一切好像很沉闷，没有这座城市他应有的生气和活力。
而在离西直门不远的北方铁道学院，却是另一番热闹景象。最近几天，因为来了新同学，社团在忙着招新。大二大三的老生们忙忙碌碌地做着工作，似乎在显示他们的干练，他们早已摆脱了当初初入学时候的稚气和拘谨，俨然一派主人公的架势。老生们，似乎很兴奋，伴随着上一届社团领导们卸任，他们终于也踏上了领导者的位置。从被领导变作领导，是该兴奋些。
社团招新，跟大学选择专业有些相仿，有冷门和热门。学生会照例是最大热门，因为进了这里，并且谋得一官半职，好似让人身价倍增，何况学生会的人也是最容易接触校领导和辅导员的。所以每年学生会之类手握重权的机构招新总是门庭若市。而像文学社这类社团基本上是少人问津的地方，这里油水最少。何况像北方铁道学院这样的理工学校，文学社更像个异类。文学不象摄影、运动，这年头你可以说爱好摄影、爱好篮球网球，但你说你爱好文学，肯定有好事者说你丫又装逼了。大多人不是好事者，但他们口头不说，或许还会恭维你，因为他们只心底里不屑。
铁道学院的校园树荫葱葱，微风吹过处树叶婆娑，无论驻足或者漫步，也是好不惬意好不清凉，与外面相比好似两个世界。招新场地是专门的活动场地，一眼扫去，人是一堆接一堆的。犄角之处人头攒动，围观者甚多，竟然是文学社，这真是破天荒。传言，今年的新任社长是个大美女，去年任副社长，今年晋升为正的。来围观的同学大半也是慕她名而来，所谓醉翁之意不在酒，在乎美色之间也。看来，人性逐利，也逐色。 而且今年文学社的宣传力度很大，当然得益于新任社长的功劳。
文学社这边有五个人负责招新，社长范琪－所谓的美女社长。琪者，玉也，范琪本人长相确实也是珠润玉滑，略施粉黛，一副干练模样。张小瑷，宣传委员，干练程度不输社长，日常宣讲活动通常是她负责。赵一书，组织委员，在社内任劳任怨，大家都叫他小书子，名义上是组织委员，实际上却经常被前两位使唤来召唤去的。他们三个负责宣讲和拉人，另外还有两位男同学在负责散发传单和登记。
“小书子，李晓凡不是说过来么，怎么还没到？”范琪问道。
“他跟方芳出去了，可能晚点来吧”小书子回道。
小瑷说道：“他俩最近走的挺近啊，是不是有情况啊。”
范琪嘴角抽动，似乎有丝不悦之意。“社团就属他俩最不积极了，尤其是李晓凡，吊儿郎当的，每次活动不是迟到就是忘记参加”范琪抱怨道。
社团总共十来个人，每次活动，并不总是人员齐聚的。此次招新，本来李晓凡和方芳答应过来的，可是早上时候方芳非拉着李晓凡出去，李晓凡问是干嘛去，方芳说去趟火车站接我一个姐们，中午前就能回来。李晓凡无奈就跟着去了，他本以为是去帮人抗行李的。可是，到了车站，事情却大自己出乎意料了。
俩人到得北京火车站，看着人群涌动，熙熙攘攘。现在虽非春运时节，旅客之多，还是非同凡响。火车站是个众生百相的地方，各色人等，汇聚于斯。有背包的，有拎包的，还有扛包的。有衣着光鲜者，也有衣衫褴褛者。车站似乎是北京城的缩影。京城内穷富荣辱，有人一掷千金，有人只能夜宿天桥。车站内，有人走vip通道、贵宾候车室候车，有人广场席地而坐。
两人正向出站口走着，忽然旁边过来一个小姑娘，八九岁模样，看上去很瘦弱，不过挺干净的模样。小姑娘向他们面前递出一张牌子，上面写着“我是聋哑人，希望爱心人士买下红绳，15元。”然后，小姑娘从挎包里掏出一副红绳，边比划边往方芳手里塞。方芳一看，就识破这骗术，有些恼怒，甩开了小姑娘的手。小姑娘一愣，转而奔向李晓凡。李晓凡本也是想躲的，奈何小姑娘拉住了，对他又摸脸又摸手，弄的李晓凡好不自在。他看向小姑娘，眉清目秀、大眼睛炯炯有神，始终微笑着，毫无城府的微笑，孩童的笑，只是面有饥色有点营养不良的感觉。李晓凡不忍心，收了红绳，打开钱包，只有张20元的零钱，递给了那小姑娘。小姑娘向两人鞠一躬，又冲晓凡一笑，转身就走了。李晓凡注视着小姑娘的背影，看她又找到了新的目标，只是新目标甩开手，狠狠瞪了她一眼，小姑娘只好笑着继续下一个目标。晓凡突然内心一阵酸痛。
这酸痛还未及扩散，就听方芳说到：“你何苦给她，明明就是个骗子，这么小就出来行骗。”
晓凡一笑，道：“只是觉得她可怜，这么小就被人指使出来干这个。如果她没有收获，说不定回去了被打也有可能。”
“你还挺有同情心。”方芳道。
“嘿嘿，所谓勿以善小而不为，无为恶小而为之。我觉得力所能及的，但凡能帮就帮，一般我碰见乞丐，我也会给几个钱的。”晓凡笑道。
“现在的乞丐还真看不上你那几个钱。”方芳冷笑道，“北京城甚至全中国，有灾有难，需要同情的人多了去了，你同情得过来嘛。以前，我也想过为社会尽责，为道义担当，甚至去西部去山区。但我现在想明白了，我为什么要为政府的制度、政策的过失买单，而某些官员们号召我们去奉献一切，他们却莺歌燕舞、公款吃喝。”
李晓凡听了有点愣，想不到这个女生竟有这些个道理，难怪法律系的。
